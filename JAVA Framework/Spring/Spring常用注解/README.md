# 类型类

类型类注释包括controller、service等，需要重点了解

其中component和bean注解的区别如下：

@Component注解在类上使用表明这个类是个组件类，需要Spring为这个类创建bean。

@Bean注解使用在方法上，告诉Spring这个方法将会返回一个Bean对象，需要把返回的对象注册到Spring的应用上下文中。
