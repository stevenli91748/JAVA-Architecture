


# spring-boot-starter-test 启动器能引入这些 Spring Boot 测试模块：

        JUnit：Java 应用程序单元测试标准类库。
        Spring Test & Spring Boot Test：Spring Boot 应用程序功能集成化测试支持。
        Mockito：一个Java Mock测试框架。
        AssertJ：一个轻量级的断言类库。
        Hamcrest：一个对象匹配器类库。
        JSONassert：一个用于JSON的断言库。
        JsonPath：一个JSON操作类库。
